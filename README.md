## :dart: Description ##

ATM exercise is a project from week 15 of the MITXPRO "Full-Stack with MERN" certification, in this project we create the interface of an ATM with React.
To make this project use [Create React App](https://create-react-app.dev/)

## :toolbox: Installation ##

Clone Repository

```bash
$ git clone https://github.com/arturomsoberanes/atm.git
```
Change the name of directory

```bash
$ mv atm atm-2
```
Install Create React App.

```bash
$ npx create-react-app my-app 
```

Replace the files for the app.

```bash
$ cp -r atm-2/ my-app/ 
```

## :joystick:	How to Run ##

Access in directory

```bash
$ cd my-app
```

Execute "start"

```bash
$ npm start
```
